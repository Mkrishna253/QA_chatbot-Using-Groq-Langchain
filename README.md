# QA_chatbot-Using-Groq-Langchain

Chatbot using Groq LLMs, langchain framework and Streamlit for an interactive user interface.

## Features
- Enter YOUR GROQ API KEY in input field of Groq API Key field.
- Select a model from the given models (e.g., llama-3.1-8b-instant, Qwen3-32B).
- Validates API and invoke LLMs with user queries.
- Stores history with session states.
- High speed and accurate response
